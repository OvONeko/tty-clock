
## tty-clock

Show a clock in your tty.  
```
usage: tty-clock [-hsc1] [-C Color]
    -h show this oage
    -s show second
    -c center the clock when start
    -1 show once
    -C <Color> Change Color: 0~F

keyboard shortcuts:
    Q: quit tty-clock
    C: center the clock
    M: Locate the clock at topleft
    S: show/unshow the second
    E: Change color
```
